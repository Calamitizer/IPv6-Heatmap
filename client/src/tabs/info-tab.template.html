This is a service providing a heatmap visualization of all reported IPv6 addresses in the world. When you pan or zoom the map on the left, the data for the displayed region are automatically downloaded to populate it.

In addition, this app exposes a public API node to directly retrieve IPv6 data. See the API tab () for specifications.
